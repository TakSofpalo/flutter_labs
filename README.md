To-Do List: Менеджер задач с погодой
Название проекта и краткое описание
To-Do List: Менеджер задач с погодой — минималистичное мобильное приложение для управления повседневными задачами с интегрированным модулем отображения текущей погоды. Приложение помогает эффективно планировать день, объединяя в одном интерфейсе список дел и актуальный прогноз. Данные задач сохраняются локально, что обеспечивает мгновенный доступ и работу без необходимости постоянного подключения к сети.

Основные функции
-Управление задачами:
	Быстрое добавление новых дел через текстовое поле.
	Отметка выполнения с визуальным изменением (заполненный чекбокс, перечёркнутый серый 	текст).
	Удаление задач интуитивным свайпом влево/вправо.
-Погодный виджет:
	Автоматический запрос к погодному API при запуске.
	Отображение города, текущей температуры, описания состояния, иконки погоды и параметра 	«ощущается как».
	Индикатор загрузки на время получения данных.
-Локальное хранение: Все задачи автоматически сохраняются в памяти устройства и полностью 	восстанавливаются после перезапуска приложения.
-Информационный раздел: Экран с версией приложения, кратким описанием и контактами разработчика.

--------Структура экранов-----------------------
Приложение состоит из двух основных экранов, оптимизированных под мобильное взаимодействие:
Главный экран (HomeScreen)
	AppBar: Заголовок «Мои задачи».
	Блок погоды: Расположен в верхней части. Содержит название города, температуру, описание, 	«ощущается как» и тематическую иконку. При первом запуске отображается состояние «Загрузка 	погоды...».
	Панель ввода: Текстовое поле с кнопкой [ + ] для мгновенного добавления задачи.
	Список задач: Вертикальный ListView. Каждый элемент содержит:
	Чекбокс для смены статуса.
	Текст задачи.
	Поддержку свайпа с появлением кнопки/зоны удаления.
	Визуальное разделение активных и выполненных задач.
-Экран «О приложении» (AboutScreen)
	Центрированная иконка приложения.
	Название и версия: To-Do List v1.0.
	Краткое описание: «Приложение для управления задачами с функцией погоды».
-Блок информации о разработчике.

------Навигация------
Механизм: Реализована через Bottom Navigation Bar с двумя фиксированными вкладками: Задачи и О приложении.
Логика работы:
По умолчанию при запуске открывается Главный экран.
Переключение между экранами происходит мгновенно при нажатии на соответствующую вкладку в нижней панели.
Стек навигации поддерживает стандартные мобильные паттерны: плавные переходы, сохранение состояния списков при переключении вкладок и возврат к последнему активному экрану.
